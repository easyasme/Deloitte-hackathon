**STEP 1 — Load data:**

df = pd.read_csv(INPUT_FILE, low_memory=False)

**STEP 2 — Sort for correct temporal ordering (critical for groupby shift):**
```python
df = df.sort_values(['ZIP', 'Category', 'Year']).reset_index(drop=True)
```

**STEP 3 — One-hot encode categories BEFORE lag computation (per D-10, D-11):**
Categories: HO, CO, DT, RT, DO, MH, NA — create binary columns `cat_HO`, `cat_CO`, `cat_DT`, `cat_RT`, `cat_DO`, `cat_MH`, `cat_NA`.
```python
categories = ['HO', 'CO', 'DT', 'RT', 'DO', 'MH', 'NA']
for cat in categories:
    df[f'cat_{cat}'] = (df['Category'] == cat).astype(int)
```

**STEP 4 — Compute one-year LAG features via groupby shift (per D-03 through D-06):**
Lag columns list:
- `Avg Fire Risk Score`
- `Earned Exposure`
- `Earned Premium`
- `CAT Cov A Fire -  Incurred Losses`
- `Non-CAT Cov A Fire -  Incurred Losses`
- `Avg PPC`

```python
lag_features = [
    'Avg Fire Risk Score',
    'Earned Exposure',
    'Earned Premium',
    'CAT Cov A Fire -  Incurred Losses',
    'Non-CAT Cov A Fire -  Incurred Losses',
    'Avg PPC',
]

for col in lag_features:
    df[f'{col}_lag1'] = df.groupby(['ZIP', 'Category'])[col].shift(1)
```

**STEP 5 — Compute EXPANDING window statistics (per D-07 through D-09):**
These capture long-term risk and exposure history per ZIP. Compute AFTER sorting and BEFORE filtering out 2018 rows (2018 seeds the expanding window).

```python
# Expanding mean of fire risk — shift(1) excludes current year to prevent leakage
df['expanding_fire_risk_mean'] = df.groupby(['ZIP', 'Category'])['Avg Fire Risk Score'].transform(
    lambda x: x.expanding().mean().shift(1)
)

# Expanding std of fire risk — may be NaN for single-observation groups (acceptable)
df['expanding_fire_risk_std'] = df.groupby(['ZIP', 'Category'])['Avg Fire Risk Score'].transform(
    lambda x: x.expanding().std().shift(1)
)

# Expanding mean of earned exposure
df['expanding_exposure_mean'] = df.groupby(['ZIP', 'Category'])['Earned Exposure'].transform(
    lambda x: x.expanding().mean().shift(1)
)
```

**STEP 6 — Add COVID-19 year indicator (per D-12, D-13):**
```python
df['is_covid_year'] = (df['Year'] == 2020).astype(int)
```

**STEP 7 — Remove 2018 rows (no prior year available for lags — per D-06):**
2018 rows served as lag source but have no lagged features of their own.
```python
df_final = df[df['Year'] != 2018].copy()
print(f"Final shape (2019+2020 only): {df_final.shape}")
print(f"Year distribution:\n{df_final['Year'].value_counts().sort_index()}")
```

**ASSERTIONS to validate:**
```python
# Lag features exist and are not NaN for 2019 and 2020 rows
for col in lag_features:
    lag_col = f'{col}_lag1'
    nan_count = df_final[lag_col].isna().sum()
    # NaN only from 2018 removal, not from missing prior year
    print(f"  {lag_col}: {nan_count} NaN (should all be from 2018 exclusion)")
    assert df_final[df_final['Year'].isin([2019, 2020])][lag_col].notna().all(), \
        f"Lag feature {lag_col} has NaN in target years"

# Category columns sum to 1 per row
cat_cols = ['cat_HO', 'cat_CO', 'cat_DT', 'cat_RT', 'cat_DO', 'cat_MH', 'cat_NA']
assert df_final[cat_cols].sum(axis=1).eq(1).all(), "Category columns do not sum to 1 per row"

# COVID flag is 1 for 2020, 0 for 2019
assert df_final[df_final['Year'] == 2019]['is_covid_year'].unique()[0] == 0
assert df_final[df_final['Year'] == 2020]['is_covid_year'].unique()[0] == 1
```

**STEP 8 — Save output:**
```python

print(f"\nOutput saved: {OUTPUT_FILE}")
print(f"Rows: {df_final.shape[0]}, Columns: {df_final.shape[1]}")
```
