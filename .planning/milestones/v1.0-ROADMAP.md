# Milestone v1.0: Insurance Premium Time Series Prediction

**Status:** ✅ SHIPPED 2026-04-10
**Phases:** 1-5
**Total Plans:** 9

## Overview

Predict 2021 insurance premiums for ~1,600 California ZIP codes using 2018-2020 historical panel data. Stack: Python + pandas + statsmodels (Panel FE) + LightGBM. Core pattern: Global Model with Entity Features (pool across ZIPs, not per-ZIP models).

## Milestone Summary

**Decimal Phases:** None

**Key Decisions:**
- Stack: Python 3.11+, pandas, statsmodels, LightGBM, scikit-learn
- Architecture: Global Model with Entity Features (pool across ~1,600 ZIPs, not per-ZIP models)
- Validation: Temporal split (train 2019, validate 2020) — NO random split
- COVID-19: 2020 flagged as structural break — downweighted at evaluation stage

**Issues Resolved:**
- pandas 3.0 NaN preservation in groupby aggregation (all-NaN groups preserved)
- AbsorbingEffectError with single-period panel data (switched to Pooled OLS)
- Singular matrix in clustered SE (switched to robust SE)
- Category dummy multicollinearity (dropped cat_ columns)
- ZIP alignment bug in fire risk ablation (verified with assertion)

**Issues Deferred:**
- Tweedie GLM for zero-inflated premium distribution (MODEL-05 in v2)
- Spatial clustering of neighboring ZIP codes (v2)
- Hierarchical/mixed-effects model (v2)
- Negative prediction clipping (40.7% of predictions are negative)

**Technical Debt Incurred:**
- LightGBM predictions not persisted to disk (regenerated inline in ensemble script)
- VERIFICATION.md not created for phases 1-3 (pre-GSD enforcement historical gap)
- Requirements-completed field absent from phases 1-3 SUMMARY.md frontmatter
